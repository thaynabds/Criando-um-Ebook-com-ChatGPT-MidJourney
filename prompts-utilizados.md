# 📋 Prompts Utilizados na Criação do eBook
## Desafio DIO — Criando um eBook com IA Generativa
### Autora: Thayná Batista da Silva

---

## 🎯 1. PROMPT — Definição do Tema e Título

```
Você é um especialista em criação de conteúdo educacional para tecnologia.
Crie 5 sugestões de títulos poderosos para um eBook voltado para iniciantes
em Inteligência Artificial, com foco prático e motivador.
Os títulos devem:
- Ser diretos e instigantes
- Transmitir transformação ou benefício claro
- Ter no máximo 10 palavras
- Ser adequados para estudantes de tecnologia no Brasil
```

**Título escolhido:** *"IA na Prática: Do Zero ao Primeiro Projeto com ChatGPT"*

---

## ✍️ 2. PROMPT — Estrutura e Sumário do eBook

```
Você é um designer instrucional especialista em eBooks técnicos.
Crie um sumário completo para um eBook de 5 capítulos com o título:
"IA na Prática: Do Zero ao Primeiro Projeto com ChatGPT"

Público-alvo: estudantes de ADS (Análise e Desenvolvimento de Sistemas),
iniciantes em IA, sem conhecimento prévio em programação avançada.

Para cada capítulo, forneça:
- Título do capítulo
- 3 a 4 tópicos principais
- Uma frase de introdução motivadora

Formato: estrutura clara com marcadores e hierarquia visual.
```

---

## 📝 3. PROMPT — Geração do Conteúdo por Capítulo

```
Atue como um professor de tecnologia didático e apaixonado pelo tema.
Escreva o Capítulo 1 do eBook "IA na Prática: Do Zero ao Primeiro Projeto com ChatGPT".

Título do capítulo: "O Que é Inteligência Artificial e Por Que Você Precisa Conhecer"

Regras de escrita:
- Linguagem acessível e encorajadora, como um mentor falando com um aluno
- Use analogias do cotidiano para explicar conceitos técnicos
- Inclua ao menos 1 exemplo prático por tópico
- Termine cada seção com uma dica rápida em destaque
- Tamanho: entre 400 e 500 palavras
- Tom: profissional, mas próximo e motivador
```

*(Repita este prompt adaptando o número e título para cada capítulo)*

## 🎨 5. PROMPT — Paleta de Cores e Identidade Visual

```
Você é um designer gráfico especialista em identidade visual para publicações digitais.
Sugira uma paleta de cores profissional para um eBook de tecnologia com IA.

Requisitos:
- 1 cor primária (uso principal)
- 1 cor secundária (destaques)
- 1 cor de fundo (dark mode)
- 1 cor de texto principal
- 1 cor de texto secundário/muted
- Justifique cada escolha
- Forneça os códigos HEX
- A paleta deve transmitir: inovação, confiança e modernidade
```

**Paleta adotada:**
| Função | Cor | HEX |
|---|---|---|
| Primária | Roxo vibrante | `#7C3AED` |
| Secundária | Ciano elétrico | `#06B6D4` |
| Fundo | Preto profundo | `#0F0F1A` |
| Texto principal | Branco suave | `#F8FAFC` |
| Texto secundário | Cinza lilás | `#94A3B8` |

## 💡 6. PROMPT — Página de Agradecimentos

```
Escreva uma página de agradecimentos para o eBook "IA na Prática"
de forma pessoal, calorosa e inspiradora.
A autora é estudante de ADS (Análise e Desenvolvimento de Sistemas)
no primeiro ano da faculdade, apaixonada por tecnologia e IA.
Mencione: a comunidade DIO, professores, família e leitores.
Tamanho: 1 parágrafo curto (máx. 80 palavras).
Tom: grato, autêntico e motivador.
```

## 🔧 7. PROMPT — Revisão e Coesão Final

```
Revise o texto a seguir para um eBook técnico-educacional sobre Inteligência Artificial.
Verifique e corrija:
1. Coerência e coesão entre parágrafos
2. Linguagem acessível (evite jargões sem explicação)
3. Consistência no tom (motivador e didático)
4. Pontuação e gramática em português brasileiro
5. Sugira melhorias de fluidez sem alterar o conteúdo essencial
```
