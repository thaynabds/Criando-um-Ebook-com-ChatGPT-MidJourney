## 📌 Sobre o Projeto

Este repositório documenta todo o processo de criação de um **eBook profissional sobre IA Generativa**, desenvolvido como Desafio de Projeto no Bootcamp **Bradesco — GenAI & Dados** da [DIO](https://dio.me/).

O projeto vai além de simplesmente gerar texto com IA: ele aplica **Engenharia de Prompt** de forma estruturada para produzir um material educacional com identidade visual única, conteúdo coeso e diagramação profissional.

## 🛠️ Como Foi Feito

O eBook foi criado inteiramente com ferramentas de IA, seguindo um fluxo estruturado:

```
  💡 Ideia & Tema
       │
       ▼
  🧠 ChatGPT (GPT-4)
  Geração de título, sumário,
  conteúdo por capítulo e revisão
       │
       ▼
  🖼️ Midjourney v6
  Criação da capa e imagens
  ilustrativas de cada capítulo
       │
       ▼
  🎨 PowerPoint / Canva
  Diagramação, layout e
  identidade visual
       │
       ▼
  📄 Exportação em PDF
  eBook finalizado e publicado
```

### 🔧 Ferramentas Utilizadas

| Ferramenta | Finalidade |
|---|---|
| **ChatGPT (GPT-4)** | Geração e revisão de todo o conteúdo textual |
| **Midjourney v6** | Criação da capa e imagens dos capítulos |
| **PowerPoint / Canva** | Diagramação, layout e formatação visual |
| **Engenharia de Prompt** | Estruturação dos comandos para resultados precisos |

## 💡 Prompts Utilizados

Os prompts completos estão documentados em [`prompts/prompts-utilizados.md`](./prompts/prompts-utilizados.md).

A estratégia de Engenharia de Prompt aplicada neste projeto seguiu os princípios:

**1. Definição de Persona**
> Atribui um papel especialista ao modelo antes de cada tarefa ("Você é um professor de tecnologia didático...")

**2. Contexto Rico**
> Fornece informações sobre público-alvo, tom desejado e restrições claras

**3. Formato de Saída Especificado**
> Define tamanho, estrutura e estilo esperados na resposta

**4. Iteração e Refinamento**
> Cada seção passou por ao menos uma rodada de revisão com prompt dedicado

## 🚀 Como Usar Este Repositório

1. Acesse [`prompts/prompts-utilizados.md`](./prompts/prompts-utilizados.md) para ver todos os prompts
2. Adapte os prompts para o seu tema de eBook
3. Use o ChatGPT para gerar o conteúdo seguindo a estrutura documentada
4. Use o Midjourney (ou outra ferramenta de imagem) para criar as ilustrações
5. Diagrame no PowerPoint ou Canva usando a paleta de cores sugerida
6. Exporte em PDF e publique no seu repositório!
